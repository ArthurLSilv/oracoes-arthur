# 🚀 Início Rápido

## 1. Preparação

Certifique-se de ter Node.js instalado (https://nodejs.org/)

## 2. Instalação das Dependências

Abra o terminal na pasta do projeto e execute:

```bash
npm install
```

## 3. Iniciar o Servidor

```bash
npm start
```

Você verá a mensagem:
```
Servidor rodando em http://localhost:3000
Conectado ao banco de dados SQLite
```

## 4. Acessar a Aplicação

Abra seu navegador e vá para:
```
http://localhost:3000
```

## Dúvidas Frequentes

**P: Como desligar o servidor?**
R: Pressione `Ctrl + C` no terminal

**P: Como acessar de outro computador?**
R: Use o IP da máquina: `http://[seu-ip]:3000`

**P: Como resetar os dados?**
R: Delete o arquivo `database.db` e reinicie o servidor
